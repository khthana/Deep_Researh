# Cyber Security Analyst Competencies - Proficiency Levels

## 1. Network Monitoring and Incident Investigation

### Basic
- Familiar with common network monitoring tools and SIEM systems
- Can identify basic security threats with guidance
- Understands the principles of digital forensics and incident response

### Intermediate
- Proficient in using multiple network monitoring tools and SIEM systems
- Can independently detect and analyze common security threats
- Applies digital forensics techniques in straightforward investigations

### Advanced
- Expert in advanced features of network monitoring tools and SIEM systems
- Can detect, analyze, and respond to complex and novel security threats in real-time
- Leads forensic investigations and develops new investigative techniques

## 2. Security Architecture and Implementation

### Basic
- Understands basic security measures and controls
- Can assist in maintaining firewalls and IDS/IPS under supervision
- Familiar with common encryption technologies and standards

### Intermediate
- Implements various security measures and controls independently
- Maintains and troubleshoots firewalls, IDS/IPS, and other security solutions
- Applies encryption technologies across different scenarios

### Advanced
- Designs comprehensive security architectures
- Optimizes and integrates advanced security solutions across platforms
- Evaluates and implements cutting-edge encryption and security technologies

## 3. Vulnerability Assessment and Penetration Testing

### Basic
- Conducts basic vulnerability scans using standard tools
- Understands the principles of penetration testing
- Can interpret simple vulnerability reports

### Intermediate
- Performs thorough vulnerability assessments using multiple tools
- Conducts standard penetration tests on systems and networks
- Prioritizes vulnerabilities and suggests remediation strategies

### Advanced
- Designs custom vulnerability assessment methodologies
- Performs advanced penetration testing, including social engineering and physical security tests
- Develops complex remediation strategies for high-risk vulnerabilities

## 4. Root Cause Analysis and Incident Investigation

### Basic
- Understands the principles of root cause analysis
- Can assist in basic incident investigations
- Documents findings using standard templates

### Intermediate
- Conducts root cause analysis for common security incidents
- Investigates and traces attack vectors independently
- Provides detailed remediation recommendations

### Advanced
- Leads complex root cause analyses for major security breaches
- Develops new investigative techniques for novel attack vectors
- Creates comprehensive security improvement strategies based on investigations

## 5. Policy and Procedure Development

### Basic
- Familiar with common industry standards and regulations
- Assists in drafting basic security policies and procedures
- Understands the importance of aligning security with business objectives

### Intermediate
- Develops security policies and procedures aligned with specific standards
- Implements security practices that support business strategies
- Contributes to the adaptation of policies for different business units

### Advanced
- Creates comprehensive security frameworks tailored to organizational needs
- Ensures seamless integration of security practices into business strategies
- Leads the development and implementation of organization-wide security policies

## 6. Continuous Learning and Adaptation

### Basic
- Stays informed about common cyber threats
- Participates in required training and development activities
- Adapts to new tools and technologies with guidance

### Intermediate
- Actively researches emerging cyber threats and technologies
- Seeks out learning opportunities and applies new knowledge
- Quickly adapts to new security technologies and methodologies

### Advanced
- Anticipates future trends in cybersecurity
- Contributes to the field through research, presentations, or publications
- Leads the adoption of innovative security technologies and practices

## 7. Incident Response and Mitigation

### Basic
- Understands basic incident response protocols
- Assists in incident response efforts under supervision
- Follows established procedures during security incidents

### Intermediate
- Independently handles common security incidents
- Applies incident response frameworks effectively
- Coordinates response efforts within a team

### Advanced
- Leads complex, large-scale incident response efforts
- Develops and improves incident response strategies and frameworks
- Makes critical decisions under pressure to minimize damage during major breaches

## 8. Reporting and Communication

### Basic
- Creates basic reports using standard templates
- Communicates security concepts to technical peers
- Presents simple data visualizations

### Intermediate
- Produces comprehensive technical reports
- Effectively communicates security issues to various stakeholders
- Creates clear data visualizations for different audiences

### Advanced
- Develops executive-level reports and presentations
- Articulates complex security concepts to non-technical senior management
- Creates sophisticated data visualizations that drive decision-making

## 9. Risk Management

### Basic
- Understands basic risk management concepts
- Assists in risk assessments using established frameworks
- Familiar with common risk mitigation strategies

### Intermediate
- Conducts risk assessments independently
- Develops and implements risk mitigation strategies
- Quantifies risks and performs basic cost-benefit analyses

### Advanced
- Leads enterprise-wide risk management initiatives
- Develops custom risk assessment methodologies
- Provides strategic guidance on risk appetite and tolerance to senior leadership

## 10. Threat Intelligence

### Basic
- Understands the concept of threat intelligence
- Uses basic threat intelligence tools and platforms
- Recognizes common indicators of compromise (IoCs)

### Intermediate
- Collects and analyzes threat intelligence from multiple sources
- Applies threat intelligence to enhance security operations
- Produces actionable threat intelligence reports

### Advanced
- Develops sophisticated threat intelligence strategies
- Leads threat hunting initiatives based on advanced analytics
- Contributes to industry-wide threat intelligence sharing efforts
